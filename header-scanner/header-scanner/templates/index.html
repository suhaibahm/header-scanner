HeaderCheck

a small tool that checks if your site sends the security headers it should

What's a security header anyway?

Every time a browser loads your page, your server sends back a bunch of headers along with the actual content. Some of those headers tell the browser to lock things down - don't allow this site in an iframe, don't run inline scripts from random places, only ever connect over HTTPS, that kind of thing. A lot of sites just... don't bother setting them. This checks for the common ones and tells you what's missing and why it matters.
